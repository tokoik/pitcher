## 2. 使用方法

- マウスの左ボタン：クリックした位置付近にめがけてボールが飛んできます．
- スペースバー：ボールが任意の方向に繰り返し放出されます．スペースバーを押す度に数が増えます．減らす方法は実装していません．
- 'o' キー：視差を広げます．
- 'c' キー：視差を狭めます．
- ESC, 'q' キー：ソフトウェアを終了します．
